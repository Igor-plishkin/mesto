# Проект: Место

------

Небольшой лэндинг для Тренировки JS

PS Update

### Функционал:
* добавление/удаление карточки
* лайк карточки
* редактирование пользователя

### Применяемые технологии:
* JavaScript
* Grid
* Flexbox
* БЭМ методология
* Webpack

### Новые фичи использованые в работе:
* Реализованы новые классы
* Webpack
* Валидация форм
